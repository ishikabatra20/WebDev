Multimedia Elements

Use <audio> and <video> for native media playback (provide controls and captions).

<video controls width="600">
  <source src="movie.mp4" type="video/mp4">
  Your browser does not support the video tag.
</video>

Best Practices

  • Keep HTML semantic and simple.
  • Do not use HTML for visual layout alone (avoid using tables for layout).
  • Use meaningful file names and clear folder structure.
  • Validate HTML using validators (W3C) to catch mistakes.

Revision Checklist (Quick)

  1. Can you create a valid HTML page from scratch?
  2. Do you know when to use semantic tags vs generic <div>?
  3. Do you remember how to build forms and tables?
  4. Do you understand basic networking terms (IP, DNS, ports)?

Final Practice Project

Create a small multi-page website (3 pages) that uses navigation, images, a contact form (it can use action="#"), and at least one table or list. Use only HTML.
