Tables

Use tables for tabular data only.

<table>
  <caption>Student scores</caption>
  <thead>
    <tr><th>Name</th><th>Score</th></tr>
  </thead>
  <tbody>
    <tr><td>Priya</td><td>95</td></tr>
  </tbody>
</table>

Forms

Collect user data with <form> and input controls.

<form action="/submit" method="post">
  <label>Name: <input type="text" name="name"></label><br>
  <label>Email: <input type="email" name="email"></label><br>
  <label>About: <textarea name="about" rows="4" cols="40"></textarea></label><br>
  <input type="submit" value="Send">
</form>

Input Types to Know

  • text, email, password, number, date, checkbox, radio, file, submit
  • textarea and select for longer input and choices.

Practice Tasks

  2. Create a small table of 3 rows and 2 columns with a caption.
