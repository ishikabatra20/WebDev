Text formatting, Lists, Links & Images

Text & Inline Tags

  • <strong> and <em> for emphasis.
  • <code> for code fragments, <small> for fine print, <mark> to highlight.

Lists

Ordered list:

<ol>
  <li>First</li>
  <li>Second</li>
</ol>

Unordered list:

<ul>
  <li>Apple</li>
  <li>Banana</li>
</ul>

Links & Anchors

Use <a href="...">link text</a>. For internal anchors use id.

<a href="https://example.com">Visit example</a>
<a href="#summary">Jump to summary</a>

<h4 id="summary">Summary</h4>

Images

Always include alt for accessibility.

<img src="image.png" alt="Description of image">

Practice Tasks

  1. Create a page with an ordered list and an internal anchor.
  2. Add two images with descriptive alt text.
