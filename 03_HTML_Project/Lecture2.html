HTML Basics: Tags, Elements & Document Structure

◀ Index | ◀ Prev | Next ▶

Key Concepts

  • HTML document: starts with <!doctype html> and has <html>, <head>, <body>.
  • Elements: start tag, content, end tag (e.g., <p>Hello</p>).
  • Void elements: elements without end tags (e.g., <br>, <img>).
  • Attributes: provide extra info (e.g., src, alt, href, id, class — although class used here only as attribute, no CSS).

Common Structural Tags

  • <header>, <nav>, <main>, <article>, <section>, <footer>.
  • <h1>..<h6> for headings.
  • <p> for paragraphs, <hr> for separators.

Example: minimal HTML page

<!doctype html>
<html lang="en">
<head>
  <meta charset="utf-8">
  <title>My Page</title>
</head>
<body>
  <h1>Hello</h1>
  <p>This is a paragraph.</p>
</body>
</html>

Practice Tasks

  1. Create a page with 3 headings and 2 paragraphs. Save and open in a browser.
  2. Try adding an image with <img src="..." alt="..."> (use local file or URL).
